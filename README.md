# 해상 기상특보 서비스

대한민국 해상 기상특보 조회 웹 서비스

## 기능

- 🌊 실시간 기상특보 조회
- 🗺️ 해구별 기상 정보
- 🌡️ 부이 관측 데이터
- 📅 물때 정보

## 기술 스택

- Frontend: HTML, CSS, JavaScript
- Backend: Netlify Functions (Serverless)
- Hosting: Netlify

## API 캐싱 전략

| 데이터 | 호출 주기 | 캐싱 |
|--------|----------|------|
| 기상특보 | 온디맨드 | 2분 |
| 해구별 기상 | 00:30, 01:30, 12:30, 13:30 | 영구 |
| 앞바다 기상예보 | 05:30, 06:30, 17:30, 18:30 | 영구 |
| 부이관측 | 매시 10분 | 10분 |

## 배포

Netlify를 통해 자동 배포됩니다.
